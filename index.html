Sales Quote

Submits one row per line item to the Quote List. Main items appear first; optional items follow under the same QDR #. Description & Unit Price are filled by Excel after the row lands.
Three workflows:
   • New quote — fill the form below, click Submit Quote at the bottom. The form stays filled so you can submit variations for other distributors.
   • Refresh an old quote — use the blue box at the top: type the existing Quote # and click Refresh Quote. No need to retype anything.
   • Start fresh — click Clear Form (top-right or bottom).
How: 1) Type the existing Quote # (e.g. 260521-1) in the box. 2) Click Refresh Quote. The flow clones every line item under the next revision (260521-2) with today's date — same QDR #, same parts, same prices. Use this when a customer needs an updated quote with refreshed 7-day validity, no need to retype anything.
